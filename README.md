# VaultShift FinCore Cloud Banking Platform

Production-grade mini banking system built for FinCore Bank's cloud migration initiative.

## Tech Stack
- Frontend: React, Vite, TailwindCSS
- Backend: Node.js, Express, Prisma
- Database: PostgreSQL
- Deployment: AWS, Docker, GitHub Actions

## Architecture
- Clean Architecture
- JWT Authentication & RBAC
- Containerized Services
